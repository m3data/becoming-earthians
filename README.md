# Becoming Earthians

An inquiry into how humans, technologies, and living systems sustain adaptive capacity under planetary-scale stress.

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

Site will be available at `http://localhost:4000`

## Structure

```
becoming-earthians/
├── _config.yml          # Jekyll configuration
├── _layouts/            # Page templates
├── _includes/           # Reusable components
├── _questions/          # Question collection
├── _methods/            # Method collection
├── _projects/           # Project collection
├── assets/css/          # Styling
├── orientation/         # Why this work exists
├── writing/             # Essays and notes
├── practice/            # Embodied practices
└── about/               # Contact and lineage
```
